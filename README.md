# Customer Churn Prediction

This is a Machine Learning project that predicts whether a customer will churn or not.

# Features
- Predict customer churn using ML
- Simple Streamlit UI
- Dynamic prediction graph
- Real-time user input

# Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

# How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py

#  Model Info
- Algorithm: Logistic Regression
- Accuracy: ~78%
